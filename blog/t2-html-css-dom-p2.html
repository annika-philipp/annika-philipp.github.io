Tech Talk pt 2

What is grid based design?

It means building your website on an undelying grid of rows and columns.

What are responsive webpages

Responsive webpages are build with users viewing pages from different devices with different sized screens in mind. Layout and content respond to these different sizes.

What is semantic structure?

Using semantic structure means naming tags in a way that represent their function within that page, such as header, footer, h1, h2 etc.
