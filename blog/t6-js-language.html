JavaScript Language

Describe all the use cases for:

parentheses ():
as a grouping operator: e.g. 1 + 3 * 5 = 16 vs. (1 + 3) * 5 = 20
with loops: if (x=y), for (x=0; x<5; x++), while (x<7)
with switch statements: switch (prize)
with conditional statements: if ( a < b)
with functions: they (can) contain paramenter/s when defining the function and they also contain the argument/s when functions are called.
e.g. function bikes (name, colour) {} - defining function, paramenters in ()
console.log(bikes(Trance, black)); - arguments in ()
when adding an Event Lister: document.addEventListener('click', checkForWin);
with arrow notation: () => {}

brackets []: get used when creating arrays and accessing array values. Brackets can also be used in square bracket notation to access the property of an object (instead of dot notation).

braces {}: enclose blocks of code such as functions, loops and conditional statements. They are also used with objects and enclose all the object's properties.

single quotes '' and double quotes "": get used with strings.
