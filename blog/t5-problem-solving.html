Getting blocked on a simple problem and solving it

That time when I wrote one of my previous blog posts, saved the post file, updated my index file and index image, pushed to GitHub to make it live - and nothing happened. No update. Nothing. I kept hitting the refresh button (repeatedly and more and more forcefully), cursing the slow internet which in my mind meant that my updates hadn't been received (becuase if in doubt look for the fault in anything but your own actions right?). I even checked my file on GitHub which seemed to confirm that all fault was with the internet...until I eventually noticed that the new blog post was still set in comments...Everything was there, I just hadn't uncommented it. I'm expecting more moments like this over the next few months, missing semi-colons, quotation marks, wrong brackets, you name it. Lesson learnt: pay attention to details in my own code.

What's the meaning of:

.map(): changes the elements of an array and then creates a new array with the changed elements

.filter(): returns elements from an array that fulfil a condition (evaluate to truthy) written within the block of the method (e.g. return only words of a certain length, or only numbers that are < 10 ...)

.reduce(): applies a function against an accumulator and each value of an array (left to right) to reduce it to a single value.
