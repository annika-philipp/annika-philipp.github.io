HTML, CSS and DOM

The house analogy seems really helpful when thinking about this. HTML gives the house structure (website - house; rooms - pages; beds, tables, chairs etc - objects and so on). CSS would be the design of this, how things are decorated and arranged. DOM - well I'm crap when it comes to analogies, but others have described is a a blueprint or project manager and I think that works.

What is meant by boxifying design?

Websites are designed as boxes within boxes within boxes within boxes... It breaks up the design of a whole page into smaller sections.

Box Model

The box model refers to the invisible components that make up the size of a box (margins, borders, padding and the actual content).
